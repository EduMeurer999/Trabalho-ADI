Vantagens

Essa biblioteca é capaz de produzir efeitos impressionantes com pouca codificação. Se o seu objetivo é criar um site ou aplicativo da Web envolvente e interativo, o jQuery é a ferramenta ideal para você. E para te convencer de uma vez por todas, listamos aqui algumas das diversas vantagens para utilizar essa biblioteca.
- jQuery promove simplicidade;
- Os elementos do jQuery são exibidos mesmo quando o JavaScript está desabilitado;
- O jQuery integra-se facilmente ao IDE do Visual Studio;
- jQuery faz aplicativos animados como o Flash;
- As páginas do jQuery são carregadas mais rapidamente;
- jQuery pode ser SEO amigável.

Recursos

jQuery é uma biblioteca de código aberto e possui licença dual, fazendo uso da Licença MIT ou da GNU. Foi desenvolvida para simplificar a navegação em documentos HTML, a seleção de elementos DOM, criar animações, manipular eventos, manipular CSS, desenvolver requisições http com AJAX e criação de plugins sobre ela. Permitindo aos desenvolvedores criarem diversos efeitos de forma mais simplificada e utilizando poucas linhas, até mesmo em aplicações web mais complexas.
Alguns dos recursos oferecidos pelo jQuery são:
1) Seleção e manipulação de elementos HTML.
2) Manipulação de CSS.
3) Efeitos e animações.
4) Navegação pelo DOM.
5) Ajax.
6) Eventos.

Obtendo a referência de um elemento na DOM através do Id

JavaScript
                
                    document.getElementById("elemento");
jQuery
                
                    $("#elemento");

Obtendo o elemento através da classe

JavaScript
                
                    document.getElementsByClassName("classe");
jQuery
                
                    $(".classe")

Obtendo o elemento através da tag

JavaScript
                
                    document.getElementsByTagName("body");
jQuery
                
                    $("body")

Adicionar evento de clique no elemento

JavaScript
                  
                document.getElementById("elemento").addEventListener("click", (ev)=>{
                    //função executada quando o elemento recebe um clique;
                })
jQuery
                
                    $("#elemento").click((e)=>{
                        //função executada quando o elemento recebe um clique;
                    })
                
            
                
                    $("#elemento").on("click", (e)=>{
                        //função executada quando o elemento recebe um clique;
                    });

História

jQuery é uma biblioteca JavaScript que interage com o HTML, desenvolvida para simplificar os scripts interpretados no navegador do usuário e a facilitar a vida do programador para assim dar um melhor resultado ao nosso cliente final. Criada em dezembro de 2006 no BarCamp de Nova York por John Resig, a jQuery é usada por cerca de 77% dos 10 mil sites mais visitados do mundo, sendo assim a mais popular das bibliotecas JavaScript. A jQuery é leve, seu tamanho é em torno de 30kb, ela oferece suporte a plug-ins e também conta com uma grande equipe de desenvolvedores que vem diariamente adicionando novos recursos e funções para essa biblioteca, nos disponibilizando uma grande quantidade de controles para interfaces.

Perguntas Frequentes


Porque foi utilizado o jQuery na criação desta página WEB?

Como citado nas vantagens e recursos oferecidos pelo jQuery, optamos por utilizar devido a sua simplicidade e facilidade de manuseio. Tem uma praticidade incomparável na hora de escrever os códigos e também auxilia muito na hora de trabalhar com DOM, já que sua API é mais elegante do que o JavaScript puro.
